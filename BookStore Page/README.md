# XYZ Bookstore Page
## Description

This is a simple HTML project representing a bookstore webpage. It showcases a collection of books using cards, with interactive buttons for purchasing or navigating the store. The project demonstrates the use of HTML elements, classes, ids, and semantic structure to organize content effectively.

## Features

Header and Introduction

<h1> title for the bookstore.

Introductory paragraph describing the collection of books.

Book Cards

Each book is represented inside a div with class card.

Each card contains:

<h2> for the book title

<p> for the book description

<button> with class btn for actions (Buy Now)

Cards use unique ids to target individual books (e.g., sally-adventure-book, dave-cooking-book).

Navigation Section

A paragraph prompting users to review their selections.

A container div with class btn-container that holds navigation buttons:

"View Cart" (id="view-cart-btn", class="btn")

"Checkout" (id="checkout-btn", class="btn")

Semantic Structure

Proper use of sibling and container elements:

card-container groups all book cards.

Buttons are grouped inside btn-container.
