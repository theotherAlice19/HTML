# Useful Links Page

## About This Project

This is a simple HTML page created to practice using links (`<a>` elements) and list structures (`<ul>` and `<li>`).  
The page contains a heading and a list of useful links that open in a new browser tab.

## What This Page Contains

- A main heading: "Useful Links"  
- An unordered list of links:
  - FreeCodeCamp
  - WhatsApp
  - Albashmoparmeg  
- Each link uses `target="_blank"` to open in a new tab

## How to View

1. Open `links.html` in any web browser  
2. Click on any of the links to open the respective website

## Purpose

- Practice HTML anchor tags and attributes (`href`, `target`)  
- Learn how to structure lists with `<ul>` and `<li>`  
- Understand how to make links open in a new tab

## Technologies Used

- HTML5
